Web Traffic Modeling Exploting TCP Connections Temporal Clustering through HTML-REDUCE

Autores:

Maurizio Molina

Paolo Castelli

Gianluca Foddis

Problema

El tráfico en la web está dominando internet, tanto en volumen como cantidad de usuarios que lo usan, sin embargo, el estudio estadístico de este tráfico es mediocre, provocando que las reglas de dimensionamiento del ancho de banda para la parte de acceso a la red sean empíricas. Esta situación conduce a una gran infrautilización de los recursos de red. Este articulo propone un modelo de tráfico web capaz de proporciones analíticas para el dimensionamiento del ancho de banda de las redes proveedoras y validarlo mediante simulaciones. Este modelo tiene en cuenta el comportamiento de los usuarios, el tamaño de las páginas solicitadas, la forma de transferir las páginas según HTTP.

Explicación

Primero se recopilan los encabezados de paquetes mediante Tcpdump, para que luego TCP-REDUCE extraiga los datos relacionados con cada conexión TCP establecida. Por último, se procesa la lista de conexiones mediante HTML-REDUCE aprovechando la agrupación temporal de conexiones TCP para construir una estructura de datos que contiene datos relevantes, y sobre esta estructura se realizan análisis estadísticos.

Otras Soluciones

Existen enfoques alternativos para realizar el modelado basado en mediciones del tráfico web, tal como BLT, pero se requiere mucho esfuerzo para desarrollar este tipo de herramientas, además de hardware de alto rendimiento para el análisis en línea. Otra forma es analizar los registros de proxy, pero este no está enfocado a un análisis de distribuciones exhaustivo, además de que para analizar los registros de Proxy conlleva más esfuerzo que analizar la salida de TCP-REDUCE.
